進階設定

虛擬伺服器 說明
可以讓網際網路的使用者存取到區域網路上之 WWW、FTP及其它網路服務。

特殊應用程式 說明
某些應用程式需要多個連接，像網路遊戲、視訊會議以及網路電話…等。

過濾器 說明
過濾器用於 LAN 端的電腦允許或者拒絕存取網際網路上的資料。過濾器可以使用於受限制的 Web 端阻絕使用者存取。

防火牆
防火牆用於允許或者拒絕封包資料通過機器。

DDNS
設定動態 DNS 功能。

DMZ
設定在 DMZ 下的電腦可無限制存取網際網路。

Wi-Fi Basic Setup
配置最基本的無線網路設定來建立無線網路環境。

Wi-Fi Advanced Setup
設置更詳細的無線網路設定，像Beacon Interval、Control Tx Rates和Basic Data Rates...等。

Wi-Fi Block Ack Setup
設定無線網路Block Ack機制。

Wi-Fi Security Setup
AP支援四種不同類型的無線網路安全機制，支援標準無線安全規範如: Wi-Fi Protected Access (WPA) Pre-Shared key、WPA Remote Access Dial In User Service (RADIUS)、 802.1X 和Wire Equivalence Protection (WEP)。

WPS Configuration Setup
支援WPS，輕鬆完成您的連線設定。

WPS Device Configure
選擇使用PIN碼或按鍵來設定WPS功能。

WPS Reset Configuration
恢復至原廠設定值。

AP-Client Configuration Setup
配置無線網路AP-Client功能。
